# 副檔名介紹
```
Windows 檔案名稱由有兩個部分組成，以句點分隔：第一部分是檔案名稱，第二個部分是定義檔案類型的三或四個字元組成的副檔名。 
在expenses.xlsx 中，檔案名稱的第一個部分是 expenses，副檔名為 xlsx。
```

### 如果您在檔案總管中檢視檔案時看不到檔案名稱的副檔名：
```
1.在工作列上的搜尋方塊中輸入「檔案總管」，然後搜尋結果中選取 [檔案總管]。

2.在 [檔案總管] 中 [檢視] 底下，在 [顯示/隱藏] 群組中選取 [副檔名]核取方塊。
```
