> 題目: 10010010
## step1: 判斷正負
```
以最前面的位元組判斷正負:
[1]0010010 
```
## step2-1: 執行反向運算(inverse)
```
10010010
轉換:
1's comp: 01101101
2's comp: 01101110 (1's + 1 = 2's)
```
## step2-2: 算magnitudy大小
```
二進制轉十進制:
01101110 => 110
```
## step3: 完成
```
正負號+大小:
ans = -110
```
